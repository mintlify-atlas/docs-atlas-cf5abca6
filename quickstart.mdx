Luanti is a free, open-source voxel game engine with easy modding and game creation. This guide walks you through downloading, installing, and playing for the first time.

## Download and install

<Tabs>
  <Tab title="Windows">
    <Steps>
      <Step title="Download the installer">
        Go to [luanti.org](https://www.luanti.org/) and download the latest Windows release. Two options are available:

        - **Installer (.exe)** — installs Luanti system-wide under `C:\Program Files\Minetest`
        - **Portable .zip** — extract anywhere and run directly (run-in-place mode)
      </Step>
      <Step title="Run the installer or extract the archive">
        For the installer, run the `.exe` and follow the prompts.

        For the portable `.zip`, extract the archive to a folder of your choice. No installation step is needed.
      </Step>
      <Step title="Launch Luanti">
        - **Installed**: open Luanti from the Start menu or desktop shortcut.
        - **Portable**: open the extracted folder and double-click `bin\luanti.exe`.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Linux">
    <Steps>
      <Step title="Install from your package manager">
        Luanti is available in most major distributions.

        ```bash
        # Debian / Ubuntu
        sudo apt install luanti

        # Fedora
        sudo dnf install luanti

        # Arch Linux
        sudo pacman -S luanti
        ```

        Check [luanti.org](https://www.luanti.org/) for distribution-specific packages or Flatpak/AppImage builds if your distro ships an older version.
      </Step>
      <Step title="Launch Luanti">
        Run `luanti` from your application launcher or terminal:

        ```bash
        luanti
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab title="macOS">
    <Steps>
      <Step title="Download the macOS release">
        Download the latest `.dmg` or `.zip` release from [luanti.org](https://www.luanti.org/).
      </Step>
      <Step title="Install the application">
        Open the `.dmg` and drag `luanti.app` to your Applications folder. If you downloaded a `.zip`, extract it and move the `.app` bundle to Applications.
      </Step>
      <Step title="Launch Luanti">
        Open `luanti.app` from Applications or Spotlight. On Apple Silicon Macs, macOS may ask you to allow the application the first time.
      </Step>
    </Steps>
  </Tab>
</Tabs>

## Start playing

When Luanti opens you land on the main menu. From here you can create a singleplayer world or join a multiplayer server.

### Create a singleplayer world

<Steps>
  <Step title="Select a game">
    Choose a game from the **Start Game** tab. If no games are installed, open the **Content** tab and install one from the ContentDB.
  </Step>
  <Step title="Create a new world">
    Click **New world**, enter a name, select a map generator, and click **Create**.
  </Step>
  <Step title="Play">
    Select your new world in the list and click **Play**. The world generates and you spawn in.
  </Step>
</Steps>

### Join a multiplayer server

<Steps>
  <Step title="Open the Multiplayer tab">
    Click the **Multiplayer** tab in the main menu.
  </Step>
  <Step title="Browse or add a server">
    Browse the public server list or enter a server address and port manually using **Add server**.
  </Step>
  <Step title="Connect">
    Enter your desired username and password, then click **Connect**.
  </Step>
</Steps>

<Note>
  Your username and password are registered on the server the first time you connect. There is no global account — each server maintains its own player database.
</Note>

## Basic controls overview

All controls are re-bindable in **Settings → Controls**.

| Input | Action |
|---|---|
| Move mouse | Look around |
| W / A / S / D | Move |
| Space | Jump / move up |
| Shift | Sneak / move down |
| Left mouse button | Dig / punch / use |
| Right mouse button | Place / use |
| Shift + Right mouse button | Build without using |
| Mouse wheel or 0–9 | Select hotbar item |
| I | Open inventory |
| T | Open chat |
| / | Open command prompt |
| Esc | Pause menu (singleplayer) |
| F12 | Take screenshot |

See the [full controls reference](/controls) for all key bindings including fly mode, minimap, debug overlays, and more.

## Where files are stored

Luanti separates its files into three locations:

| Location | Purpose |
|---|---|
| `bin` | Compiled executables |
| `share` | Read-only engine data and built-in games |
| `user` | Your worlds, mods, settings, and screenshots |

Your saved worlds live at `user/worlds/` and your configuration file at `user/minetest.conf`. See the [paths reference](/configuration/paths) for exact locations on each platform.
